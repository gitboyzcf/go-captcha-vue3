# go-captcha-vue3
go-captcha 在Vue3中使用样例

> [English](README.md) | 中文

## 项目设置
```
npm install
```
在项目目录中，可以运行：

### 编译和热重载开发
```
npm run serve
```

## 启动后端
```
git clone github.com/wenlng/go-captcha-example
cd go-captcha-example
go mod download
go run main.go
```
